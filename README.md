# docker-debian-11

Base image of Debian 11.

Includes:
- bash
- cron
- unzip
- wget
- nano
- composer v2
- php 7.4 fpm
- apache2
- extensions:
  - php-gd 
  - php-zip 
  - php-xml 
  - php-mysqli 
  - php-pdo 
  - php-sockets 
  - php-opcache 
  - php-mbstring 
  - php-curl

it is running on port 2222
